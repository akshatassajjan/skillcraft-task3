# 🧠 Quiz Game Application

## SkillCraft Technology Internship - Task 03

An interactive and responsive Quiz Game Application developed using HTML, CSS, and JavaScript.

## 📌 Project Description

This project is a browser-based Quiz Game Application that allows users to answer multiple-choice questions and receive instant feedback on their performance. The application tracks scores, displays results, and provides an engaging user experience.

## 🚀 Features

* Multiple Choice Questions
* Real-Time Score Tracking
* Instant Answer Validation
* Next Question Navigation
* Final Score Display
* Restart Quiz Option
* Responsive Design
* Modern User Interface
* Smooth Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Task Objective

Build a Quiz Game Application that displays multiple-choice questions, allows users to select answers, calculates scores, and provides immediate feedback.


## ⚙️ How to Use

1. Open the application.
2. Start the quiz.
3. Select the correct answer.
4. Move to the next question.
5. View your final score.
6. Restart the quiz if desired.

## 📱 Responsive Design

The application is optimized for:

* Desktop Computers
* Laptops
* Tablets
* Smartphones

## 💡 Key Learning Outcomes

* JavaScript Fundamentals
* DOM Manipulation
* Event Handling
* Score Calculation Logic
* Responsive Web Design
* UI/UX Development
